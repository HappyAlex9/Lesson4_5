# Инструкция для работы с Makrdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочками (* Текст *). Например: 
>*Вот так*

Чтобы выделить текст полужирным необходимо обрамить его звездочками (** Текст **). Например: 
>**Вот так**.
